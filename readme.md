## Development

1. Clone with `git clone`.
2. Install with `npm install` or `yarn install`.
3. Load `file:///.../index.html` in local web browser.
4. Change `index.html`.
5. Format HTML with `npm run format` or `yarn format`.
6. Refresh browser.
